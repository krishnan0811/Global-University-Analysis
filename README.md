#  Global University Analysis Dashboard

##  Project Overview

This project is an interactive **Data Analytics Dashboard** built using Python and Streamlit. It analyzes global university data to uncover insights about rankings, student distribution, faculty strength, and institutional characteristics.

The dashboard provides **clear insights with visually appealing charts** and allows users to filter data dynamically.

---

## Features

*  Interactive dashboard using Streamlit
*  KPI metrics (Total Universities, Average Students, Locations)
*  Dynamic filters (Location, University Type)
*  Multiple visualizations:

  * Top 10 Universities by Rank
  * Universities by Location
  * Student Distribution
  * University Type (Public vs Private)
  * Faculty vs Students
  * Campus Area vs Students
  * Establishment Year Trend
---

##  Tech Stack

* Python
* Pandas
* Matplotlib
* Seaborn
* Streamlit

---

##  Dataset

The dataset contains information about top universities including:

* Position (Rank)
* University Name
* Location
* Established Year
* Total Students
* Total Faculty
* Campus Area
* University Type

---

##  Key Insights

* Top-ranked universities maintain strong global reputation
* Most universities have moderate student populations
* Public and private institutions are nearly balanced
* Certain locations dominate global rankings
* Larger campuses tend to accommodate more students

##  How to Run

1. Clone the repository:

```bash
git clone https://github.com/your-username/university-dashboard.git
```

2. Navigate to the project folder:

```bash
cd university-dashboard
```

3. Install dependencies:

```bash
pip install -r requirements.txt
```

4. Run the app:

```bash
streamlit run app.py
```
